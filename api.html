API 활용

API 사용 제한
자동차 테크맵 프로젝트의 사용자 API는 다음과 같은 사용 제한이 있습니다:
  • 한 시간당 최대 5,000번 이상의 API 호출 시, 'Too Many Requests' 예외가 발생합니다.
컴포넌트 조회
GET /api/v1/component
등록된 컴포넌트 정보를 조회할 수 있는 API입니다.

Query 매개변수:
  • name: 컴포넌트명 (예: zlib)
  • License: 라이선스명 (예: MIT)
  • equalFlag: 'Y'는 정확히 일치하는 결과, 'N'(기본값)은 포함된 모든 결과
  • page: 시작 페이지 숫자 (기본값: 0)
  • size: 페이지당 데이터 개수
  • sort: 정렬 기준 항목 (예: name, id)
  • direction: 정렬 방향 ('ASC' 또는 'DESC')
사용 예제
curl --location --request GET 'https://{osori_base_url}/api/v1/oss/name/versions?searchWord=facebook'
